# site-with-sitemap

A simple static website with:
- Home page (`index.html`)
- 4 child pages (`about.html`, `services.html`, `blog.html`, `contact.html`)
- XML sitemap (`sitemap.xml`)
- Vercel config (`vercel.json`)

## Run locally

Open `index.html` directly in browser, or run a local static server.

## Test

```bash
npm test
```

## Deploy to Vercel

Import this repo into Vercel and deploy.
